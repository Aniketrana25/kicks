# Custom Art Website

A portfolio and e-commerce website for custom hand-painted artwork on shoes, denim jackets, pants, and wall art.

## Features

- Product catalog showcasing existing artwork
- Custom design tool for creating personalized items
- Shopping cart functionality
- Contact form that forwards inquiries to WhatsApp
- UPI payment integration (to be implemented)

## Setup Instructions

1. Clone this repository
2. Replace the placeholder images with your actual product images
3. Customize product details, pricing, and descriptions
4. Deploy to GitHub Pages

## Deployment to GitHub Pages

To deploy this website using GitHub Pages:

1. Push this repository to GitHub
2. Go to repository Settings
3. Navigate to Pages section
4. Select the main branch as source
5. Save the changes

Your website will be available at `https://yourusername.github.io/repositoryname/`

## Customization

- Update product images in the `images` folder
- Modify product details in `index.html`
- Adjust colors and styling in `styles.css`
- Enhance functionality in `script.js`

## UPI Payment Integration

To implement UPI payments:
1. Sign up for a payment gateway that supports UPI (e.g., Razorpay, PayU)
2. Follow their documentation to integrate with the checkout system
3. Update the checkout functionality in `script.js`

## Contact Form to WhatsApp

To connect your contact form to WhatsApp:
1. Set up a webhook or use a service like Twilio WhatsApp API
2. Update the form submission handler in `script.js`
